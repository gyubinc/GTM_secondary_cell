## 프로젝트명

GTM 기반 2차전지 국내 부족기술 및 세계 공백기술 추출

## 프로젝트 주제

현재 국내, 해외 2차전지 특허 데이터를 활용해 GTM을 기반으로 하여 2차전지 국내 부족기술 및 세계 공백기술를 추출하고 시계열 분석을 통해 기술의 기대수명을 예측하고자 함

## 기간

2022-10 ~ 2022-12
